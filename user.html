Modification de compte

La commande "user" vous permet de modifier votre compte simplement.

  • user user : modifier votre nom d'utilisateur

  • user password : modifier votre mot de passe

  • user disconnect : déconnecte votre compte

  • user delete : supprime votre compte

  • user reset : réinitialise votre clé de chiffrement
