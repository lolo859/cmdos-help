Manipulation de fichiers et navigation dans l'arborescence

CmdOS dispose de commandes pour naviguer et manipuler des fichiers et dossiers, les voici :

  • detail nom_du_fichier : affiche les propriétés d'un fichier

  • mkdir nom_du_dossier : crée un dossier

  • del nom_du_fichier/dossier : supprime un fichier / dossier

  • ren nom_du_fichier_à_renommer : renomme un fichier / dossier

  • cd nom_du_dossier / .. / ~ : ouvre le dossier / revient au dossier parent / ouvre le dossier utlisateur

  • dir : affiche le contenu du dossier actuel

  • move nom_du_fichier repertoire_de_destination : déplace des fichiers

  • copy nom_du_fichier répertoire_de_destination : copie des fichiers

  • mktxt nom_du_fichier_texte_sans_espace contenu_du_fichier : crée des fichiers textes avec un contenu

  • cat nom_du_fichier : affiche le contenu d'un fichier text
