Godmode

Le godmode est un emsemble de fonction réservé aux développeurs et utilisateurs aguerri.

Il faut être en mode admin pour y accéder.

Voici la liste des fonctions du godmode :

  • sys.variables : Cette fonction permet d'afficher la valeur des variables systèmes

  • sys.cmd.store.install.vanish_loading : Cette fonction permet de masquer le chargement lors de l'installation d'un module (désormais inutile)

  • sys.protocol.list : Cette fonction permet de voir quel protocole système fait quelle action

  • sys.protocol.execute : Cette fonction éxécute le protocole demandé (elle positionne automatiquement les arguments)

  • sys.cmd.display_split : Cette fonction permet d'afficher le split de la variable rep lors du traitement de la commande

  • sys.cmd.report : Cette fonction permet d'activer/désactiver l'enregistrement des commandes sur le serveur de support

  • sys.class.list : Cette fonction fait la liste des classes systèmes
